1. WHAT IS CRYPTGUARD?  
CryptGuard is a secure file encryption tool that uses Fernet (AES-128) symmetric encryption to protect sensitive documents, photos, and other files. It's designed for:  

✓ Families sharing personal documents  
✓ Businesses protecting confidential data  
✓ Individuals securing private files  

2. SYSTEM REQUIREMENTS  
- Windows 10/11  
- 100MB free disk space  

3. HOW TO USE  

► ENCRYPTING FILES:  
1. Click "Browse file 🔎" under "Encrypt a file here"  
2. Select any file to encrypt (documents, images, etc.)  
3. Choose where to save your encryption key (or use default)  
4. Select location for encrypted file (.enc extension)  
5. Securely share BOTH the encrypted file AND key with recipient  

► DECRYPTING FILES:  
1. Click "Browse file 🔎" under "Decrypt a file here"  
2. Select the .enc file  
3. Choose the matching .key file  
4. The decrypted file will save with original name (without .enc)  

4. SECURITY BEST PRACTICES  

✔️ DO:  
- Store keys separately from encrypted files  
- Use secure channels to share keys (Signal, ProtonMail)  
- Delete unencrypted originals after verification  
- Rename key files to avoid identification  

✖️ DON'T:  
- Reuse the same key for multiple files  
- Store keys in cloud services without encryption  
- Send keys through unencrypted email/SMS  
- Modify encrypted files manually  

5. TROUBLESHOOTING  

🔴 Error: "Decryption failed"  
→ Ensure you're using the EXACT key file created during encryption  
→ Verify the encrypted file wasn't modified after creation  

🔴 App crashes when selecting files  
→ Run as Administrator (Windows)  
→ Check file permissions  

6. DISCLAIMER  
CryptGuard provides strong encryption but cannot guarantee:  
- Protection against keylogging malware  
- Security if keys are compromised  
- Recovery of lost encryption keys  

Always maintain backups of important files.

1. CryptGuardとは？  
CryptGuardは、Fernet（AES-128）対称暗号化を使用して、機密文書、写真、その他のファイルを保護する安全なファイル暗号化ツールです。次の目的に最適です：  

✓ 個人の文書を共有する家族  
✓ 機密データを保護する企業  
✓ プライベートなファイルを保護する個人  

2. システム要件  
- Windows 10/11  
- 100MBの空きディスク容量  

3. 使用方法  

► ファイルの暗号化：  
1. 「Encrypt a file here」の下の「Browse file 🔎」をクリック  
2. 暗号化するファイルを選択（文書、画像など）  
3. 暗号化キーを保存する場所を選択（またはデフォルトを使用）  
4. 暗号化ファイルの保存先（.enc拡張子）を選択  
5. 暗号化ファイルとキーの両方を安全に受信者と共有  

► ファイルの復号化：  
1. 「Decrypt a file here」の下の「Browse file 🔎」をクリック  
2. .encファイルを選択  
3. 一致する.keyファイルを選択  
4. 復号化されたファイルは元の名前（.encなし）で保存されます  

4. セキュリティベストプラクティス  

✔️ 実行すること：  
- 暗号化されたファイルとは別にキーを保管  
- 安全なチャネルでキーを共有（Signal、ProtonMail）  
- 検証後に暗号化されていない元のファイルを削除  
- キーファイルの名前を変更して識別を避ける  

✖️ 実行しないこと：  
- 同じキーを複数のファイルで再利用しない  
- 暗号化なしでクラウドサービスにキーを保存しない  
- 暗号化されていないメール/SMSでキーを送信しない  
- 暗号化されたファイルを手動で変更しない  

5. トラブルシューティング  

🔴 エラー：「復号化失敗」  
→ 暗号化時に作成された正確なキーを使用していることを確認  
→ 暗号化ファイルが作成後に変更されていないか確認  

🔴 ファイル選択時にアプリがクラッシュする  
→ 管理者として実行（Windows）  
→ ファイルの権限を確認  

6. 免責事項  
CryptGuardは強力な暗号化を提供しますが、次のことは保証しません：  
- キーロガー型マルウェアからの保護  
- キーが危険にさらされた場合のセキュリティ  
- 失われた暗号化キーの復旧  

重要なファイルのバックアップは常に保持してください。
